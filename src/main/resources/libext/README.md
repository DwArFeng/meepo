# libext - 外部依赖文件夹

如果此项目需要加载外部依赖，请将相关依赖对应的 jar 包或 zip 包放入对应模块的根目录下。

该目录下有两个子目录，分别对应 meepo 项目的两个模块，其结构如下：

| 路径             | 用途                  |
|:---------------|:--------------------|
| confext/daemon | 用于放置 daemon 模块的外部依赖 |
| confext/poof   | 用于放置 poof 模块的外部依赖   |

*请将相关的依赖直接放入对应模块的根目录下，该项目暂不支持读取子文件夹下的依赖。*
